# Contributing to Stocked Up

Thank you for your interest in contributing to Stocked Up!

## Reporting Bugs
Please open an issue and include:
- Steps to reproduce
- Expected behavior
- Screenshots if applicable

## Suggesting Features
Open a feature request describing:
- The problem
- Your proposed solution
- Any additional context

## Pull Requests
1. Fork the repository
2. Create a feature branch
3. Commit your changes
4. Submit a pull request

Please ensure contributions align with Stocked Up's mission of privacy-first household management.
